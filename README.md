## 使用方法

你需要下载一个油猴的谷歌插件，然后将zhihu.js文件中的内容复制进去即可
顺便提一句，如果你有使用infinity插件，可以选择添加新浏览器，然后添加知乎的搜索网址，选择自定义,添加下列网址作为搜索地址
https://www.zhihu.com/search?type=content&q=%s
